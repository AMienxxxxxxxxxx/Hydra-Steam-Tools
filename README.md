## About

HydraSteamTools is a lightweight, high-performance desktop application designed to give you absolute control over your Steam library. Search any game by name or AppID, download depot manifests and LUA config scripts, automatically configure DLCs, and keep everything organized in a single, premium user interface.

## Features

*   **Fuzzy Smart Search** — Search the Steam store by name or AppID with real-time, smart punctuation-insensitive results.
*   **One-Click Install** — Download depot manifests, LUA configurations, and activate games in a single click.
*   **DLC Auto-Install** — Automatically detects and appends all available DLC content.
*   **Integrated Game Fixes** — Dedicated fixes engine loaded dynamically from Onyxaxdb for popular titles.
*   **Favorites System** — Star your favorite games to organize and batch-install them later.
*   **Steam Integration** — Restart Steam directly from the app with auto-detected installation directories.
*   **Light & Dark Themes** — Beautiful, glassmorphism-inspired themes saved automatically based on your preference.
*   **Smart Auto-Updater** — Keeps your client updated with the latest CDN optimizations.
*   **Secure Environment** — HWID-based validation with AES-256 encryption.
