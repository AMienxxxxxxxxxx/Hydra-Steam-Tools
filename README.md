## About

HydraSteamTools is a lightweight desktop application that gives you full control over your Steam library. Search any game by name or AppID, download depot manifests and Lua scripts, install DLC automatically, and manage everything from one clean interface.

---

## Features

- **Game Search** — Search the Steam store by name or AppID with real-time results
- **One-Click Install** — Download depot manifests, Lua scripts, and configure your library in a single click
- **DLC Auto-Install** — Automatically detects and installs all available DLC content
- **Favorites System** — Star any game and bulk-install your favorites later
- **Steam Integration** — Restart Steam directly from the app with auto-detected installation directory
- **Dark & Light Themes** — Toggle between themes, preference saved automatically
- **Auto-Update** — Checks for new versions on startup
- **Secure Activation** — HWID-based license verification with AES-256 encryption
